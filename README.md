# DK Furniture - E-commerce Web Application

1=Project Overview
**DK Furniture** is a professional e-commerce platform designed for a premium furniture store. The application is built using the **Laravel** framework, focusing on a clean, luxury-themed user interface and a seamless shopping experience.

2= 🚀 Current Project Status
- **Frontend Architecture:** 100% Completed (Modular Blade templates for high reusability).
- **Responsive UI:** Fully optimized for Mobile, Tablet, and Desktop.
- **Backend Planning:** Database schema (Migrations) is ready; Integration of Auth and Product CRUD is the next phase.

3=✨ Key Features
- **Dynamic Hero Slider:** Professional image slider with "Call to Action" buttons.
- **Product Categories:** Beautifully designed cards for Luxury Sofas, Dining Sets, and Bedroom Essentials.
- **Navigation System:** Includes a search bar, login/register options, and category-wise navigation.
- **Premium UI:** Custom color palette (Gold/Brown) tailored for a furniture brand.

4=🛠️ Tech Stack
- **Framework:** Laravel 11.x
- **Frontend:** Blade Templating Engine, CSS3, JavaScript
- **Design:** Fully Responsive (Mobile & Desktop friendly)
- **Backend Language:** PHP

5=📁 Project Structure
- `resources/views`: Contains all Blade templates for the UI.
- `routes/web.php`: Managed all frontend URL routing.
- `public/images`: All product and slider assets are optimized here.
